# 🧩 Foro API - Backend de Tópicos

Este proyecto es un **servicio backend REST** desarrollado con **Spring Boot** que permite administrar tópicos de un foro o sistema de discusión.  
La API expone endpoints para **crear, consultar, actualizar y eliminar tópicos**, utilizando persistencia en **MySQL** mediante **JPA/Hibernate**.

La aplicación sigue una arquitectura sencilla basada en **controladores, repositorios y entidades**, ideal para aprender desarrollo backend con Spring.

---

# 🛠 Stack Tecnológico

| Tecnología | Descripción |
|------------|-------------|
| Java 17 | Lenguaje principal del proyecto |
| Spring Boot | Framework para aplicaciones backend |
| Spring Data JPA | Persistencia de datos |
| Hibernate | ORM utilizado por JPA |
| MySQL | Base de datos relacional |
| Maven | Gestión de dependencias |
| Lombok | Reducción de código repetitivo |
| Flyway | Control de migraciones de base de datos |
